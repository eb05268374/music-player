Needs MP3 or OGG files to play (if there's no metadata attached it WILL crash out lol)

## **How to edit playlists:**
  - In the playlist view (the one that shows on first startup), press the large + button in the bottom right
  - Select one MP3 (or OGG) at a time
  - Do this for all the songs you want in the playlist

  - The little x button next to the songs file name removes it from the playlist

  - The smaller + button with the 3 lines creates a new playlist

  - The little buttons labeled with "PL" go back and forward between playlists
